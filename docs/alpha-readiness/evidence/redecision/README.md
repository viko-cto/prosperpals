# Re-Decision Evidence Folder

Store the founder-visible roll-ups used for the next formal GO / CONDITIONAL GO / NO-GO review.

## Expected artifacts
- dated dashboard snapshot(s)
- dated recommendation memo(s)

## Example filenames
- `redecision-dashboard-2026-03-28.md`
- `alpha-redecision-memo-2026-03-28.md`

## Source-of-truth rule
These files are summaries only.

They should link back to:
- interview evidence,
- hosted hardening proof,
- receipt realism traces,
- operator-readiness evidence,
- and current blockers.

If a claim cannot be supported by underlying evidence, it does not belong in the re-decision artifact.
