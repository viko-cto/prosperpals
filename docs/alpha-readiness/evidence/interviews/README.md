# Interviews Evidence Folder

Store durable participant-level and batch-level interview artifacts here.

## Expected artifacts
- 12 session logs
- 4 batch syntheses
- 1 final synthesis

## Naming convention
### Session logs
- `session-01-<participant-id>.md`
- `session-02-<participant-id>.md`
- ...

### Batch syntheses
- `batch-01-sessions-01-03.md`
- `batch-02-sessions-04-06.md`
- `batch-03-sessions-07-09.md`
- `batch-04-sessions-10-12.md`

### Final synthesis
- `final-interview-synthesis-YYYY-MM-DD.md`

## Non-negotiable fields per session
- participant ID and archetype
- first 30-second impression
- Goldie understanding
- Fin understanding
- Goldie -> Fin handoff reaction
- Daily Spending Power reaction
- ProsperCoins reaction
- simulator reaction
- receipt-review trust reaction
- first trust-drop moment
- willingness-to-return score
- strongest quote
- keep / change / cut / unresolved
