## 2. Batch 01 purpose

Batch 01 is not meant to prove product-market fit.

It is meant to create the **first disciplined contrast set** across the three core behavior shapes before wording or flow changes are made:
- **PP-B01** — budget-first
- **PP-I01** — invest-first
- **PP-M01** — mixed

If these three slots are filled carelessly, the whole interview lane becomes easier to narrate than to trust.

---

## 3. Batch 01 locked rules

These stay fixed unless repeated evidence says they are wrong:

- Batch 01 must cover **budget-first + invest-first + mixed**.
- Batch 01 should contain **at least one non-founder-network recruit** before scheduling is considered healthy.
- Batch 01 should avoid becoming three versions of the same social cluster.
- Real participant names stay **offline**; repo artifacts use aliases only.
- A slot is not ready for scheduling until source, screener fit, and rationale are recorded.
- A completed call does not count toward threshold math until the same-day session note exists.

## 7. Backup-candidate discipline

Batch 01 should not stall because one person ghosts.

Keep **at least one backup candidate per slot** offline using the same intake template above.

Rules:
- backup candidates do not go into the canonical tracker until they are the active assigned candidate,
- a backup is not a free pass to ignore source-mix discipline,
- if two Batch 01 slots depend entirely on founder-network backups, the batch is not healthy yet.
