## 🚀 Quick Decision Tree

```
What do you need?

CREATE SOMETHING?
├─ Document/File → Anthropic Skills (pdf, docx, pptx, xlsx)
├─ Code/Component → Anthropic Skills (mcp-builder, artifacts-builder)
└─ Visual/Design → Anthropic Skills (canvas-design, theme-factory)

IMPROVE PROCESS?
├─ Stuck/Blocked → when-stuck → dispatches to right technique
├─ Debugging → systematic-debugging (NO FIXES WITHOUT ROOT CAUSE)
├─ Testing → test-driven-development (NO CODE WITHOUT FAILING TEST)
└─ Planning → writing-plans → executing-plans

RESEARCH/LEARN?
├─ Web Articles → article-extractor
├─ YouTube Videos → youtube-transcript
├─ Connect Docs → tapestry
└─ Prioritize → ship-learn-next

ANALYZE DATA?
├─ CSV Files → csv-data-summarizer
├─ Excel Analysis → xlsx skill
└─ Test Cases → pypict skill

COLLABORATE?
├─ Git Workflows → using-git-worktrees, finishing-a-development-branch
├─ Code Review → requesting-code-review, receiving-code-review
└─ Team Coordination → dispatching-parallel-agents, brainstorming
```
